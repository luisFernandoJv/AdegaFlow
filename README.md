# 🍷 AdegaFlow

Sistema full-stack de gestão e controle de estoque voltado para adegas e pequenos comércios.

Desenvolvido com **React + TypeScript (Vite)** no frontend e projetado para integração com **Java (Spring Boot)** no backend, este projeto demonstra organização arquitetural, consumo de API REST, autenticação segura e estrutura escalável.

---

## 🚀 Tecnologias Utilizadas

### Frontend

- React
- TypeScript
- Vite
- Axios
- React Router
- Context API / Gerenciamento de Estado
- ESLint

### Backend (em desenvolvimento / planejado)

- Java
- Spring Boot
- Spring Security + JWT
- JPA / Hibernate
- PostgreSQL
- Swagger (OpenAPI)

## ⚙️ Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/seu-usuario/adegaflow.git
```

### 2️⃣ Instalar dependências

```bash
npm install
```

### 3️⃣ Executar o servidor de desenvolvimento

```bash
npm run dev
```
